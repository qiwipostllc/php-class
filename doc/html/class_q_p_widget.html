Инициализация объекта

Аргументы
$phonenumber Номер телефона для авторизации
$passwdstring пароль для авторизации
$pricestring Стоимость услуги, например, '100 руб.'
$testbool 1 - если тестовый доступ (в этом случае номер телефона и пароль могут быть пустые), 0 - если реальный доступ
$folderstring относительный путь к папке с файлом qpwidget.js
$mess_templatestring Текст сообщения, которое вставляется автоматически в поле с комментарием после выбора терминала QiwiPost. Например: "Выбран терминал оплаты name", где "name" - будет заменено автоматически на ID терминала. Внимание! Если в шаблоне сообщения не будет указан параметр "name", то автоматически будет использован заданный по умолчанию шаблон.
$jquerybool 1 - необходимо подключить jquery, 0 - не нужно подгружать jquery, т.к. он уже загружен на странице. Внимание! jquery необходим в любом случае, т.к. на нем работает подгружаемый js-скрипт!
$encodestring Кодировка, в которой будут загружены данные о терминалах через api QiwiPost
